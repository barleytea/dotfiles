# マークダウン表示テスト

これはマークダウンファイルのテストです。

## コードブロックの表示確認

以下はPythonのコードブロック例です：

```python
def hello_world():
    print("Hello, World!")
    return True
```

以下はJavaScriptのコードブロック例です：

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}
```

## インラインコード

インラインコードのテスト: `const x = 42;` はどのように表示されるか。

## 長文の折り返しテスト

この行は非常に長いテキストで、通常はウィンドウ幅を超えてしまうため、マークダウンファイルでは適切に折り返されることを確認します。従来の `wrap = false` 設定では読みづらかったはずです。
